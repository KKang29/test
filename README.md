# 민원데이터 자동 분류 개발 프로젝트
## 프로젝트 소개
### 인공지능을 활용하여 민원이 접수될 시, 자동으로 분류하는 서비스

## 팀원소개
### 이영훈: 인공지능 모델 개발, DB, 웹 구축 등

## 개발기간
### 2023. 3. 2 ~ 2023. 6. 20

## 개발환경
### Ubuntu 20.4
### Python 3.10
### DataBase: Mariadb 
### Web FrameWork: Flask
### DeepLearing FrameWork: Tensorflow 2.12

## 주요기능
### 민원 접수(텍스트, 오디오) → AI 분류 모델 → 17개의 카테고리에 맞춰서 자동 민원 저장 → Dash 보드 기능을 통한 민원 분포 확인
